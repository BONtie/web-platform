TestTWF Objectives

  • Bring testing and developer communities together
  • Recruit into the standards testing community
  • Teach how to write tests and log bugs
  • Add to W3C test suites
  • Keep community engaged throughout the year

2012 Events

San Francisco, CA, USA June 15 + 16

Beijing, China October 20 + 21

Paris, France October 26 + 27

San Francisco Highlights

  • ~70 Attendees, ~20 Experts
  • Talks:
    • W3C Testing: Lay of the Land
    • SVG Testing
    • CSS Testing Tools
    • How to Write W3C Tests
    • Filing the Best Bugs
  • Specs Featured: CSS Transforms, SVG, CSS Backgrounds & Borders, CSSOM
  • ~90 Tests written
  • Logged several browser, spec, and test harness bugs

Beijing Highlights

  • 134 Attendees, 21 Experts
  • Talks:
    • How to Write a W3C Test
    • W3C Testing Framework
  • Specs Featured: CSS Flex Box, CSS Backgrounds & Borders, CSSOM
  • 115 tests written
  • Submitted 12 browser bugs

Paris Highlights

  • 130 Attendees, ~35 Experts
  • Talks:
    • How to Read a Spec
    • How to Write a Reftest
    • testharness.js Overview
    • Filing the Best Bugs
  • Tests written for 13 specs: HTML5, WebApps, CSS
  • Apache Cordova suite -> W3C tests
  • ~400 tests written
