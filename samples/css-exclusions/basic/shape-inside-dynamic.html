Flowing content with CSS Exclusions: Shape Inside

With CSS Exclusions, you can set an element's content area to be an arbitrary shape using shape-inside. Use the controls in the upper right to dynamically resize the shapes and the text.

#rectangle {
    shape-inside: rectangle(0, 0, 100%, 100%, 25%, 25%);
}
#hexagon {
    shape-inside: polygon(25% 0, 75% 0, 100% 50%, 75% 100%, 25% 100%, 0 50%);
}
